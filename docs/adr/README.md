# Architecture Decision Records (ADR)

Use this folder to record important technical decisions.

## ADR Format

Each ADR should contain:

- Title
- Status (`Proposed`, `Accepted`, `Superseded`)
- Context
- Decision
- Consequences
- Alternatives considered

## Naming Convention

`0001-short-kebab-title.md`, incrementing by one.
